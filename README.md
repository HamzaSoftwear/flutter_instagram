# Flutter Instagram Clone

A Flutter-based mobile application inspired by Instagram, integrated with Firebase services.

## Features
- **User Authentication** via Firebase Authentication (Sign up / Sign in).
- **Post Upload**: Share images or videos using Firebase Storage.
- **Feed**: View posts from all users.
- **Like System**: Like posts from other users.
- **Comments**: Add comments to any post.
- **User Interaction**: Follow and engage with other users.
- Modern, Instagram-style UI for a smooth user experience.

## Requirements
- Flutter 3.x or later.
- A configured Firebase project.
- Enabled Firebase services:
  - Authentication
  - Firestore Database
  - Storage
